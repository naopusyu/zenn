## はじめに

## 環境

- PHP 8.4.1
- laravel/laravel 11.3.3
- laravel/framework 11.33.2

## db

新しいデータベースのCLIセッションを開始します。

```
php artisan db
```

| 引数 | 説明 |
| --- | --- |
| `connection` | データベース接続名を指定 |

## db:monitor

データベースの接続数を監視します。

```
php artisan db:monitor
```

| --- | --- |
| `--databases` |  |
