## はじめに

今回はconfigコマンドについて調べました。

## 環境

- PHP 8.4.1
- laravel/laravel 11.3.3
- laravel/framework 11.33.2

## config:cache

キャッシュファイルを作成します。

```
php artisan config:cache
```

## config:clear

キャッシュファイルを削除します。

```
php artisan config:clear
```

## config:publish

設定ファイルを公開します。

```
php artisan config:publish
```

| 引数 | 説明 |
| --- | --- |
| `name` | 設定ファイルの名前を指定 |

## config:show

設定ファイルの内容を表示します。

```
php artisan config:show
```

| 引数 | 説明 |
| --- | --- |
| `config` | 設定ファイルの名前を指定 |
